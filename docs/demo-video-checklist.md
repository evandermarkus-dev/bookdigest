# BookDigest — Inspelningschecklista
**Demo Video — "En dag i livet"**

Gå igenom hela listan innan du startar inspelningen. Spela in lång version först — kort version klipps från samma material.

---

## 1. Förbered innehåll

- [ ] **PDF klar:** Atomic Habits av James Clear (välkänd bok, bra för demos)
- [ ] **Rensa dashboard:** Ta bort alla tidigare uppladdade böcker så att vy är fräsch
- [ ] **Rensa localStorage:** Öppna DevTools → Application → Local Storage → radera `bookdigest_user_context` (så att questionnaire dyker upp live)

---

## 2. Datorinställningar

- [ ] **Notifikationer av:** Aktivera "Stör ej" / DND på hela datorn
- [ ] **Slack, mail, etc. stängt:** Inga popup-bubblor synliga
- [ ] **Bakgrundsbild:** Neutral/ren skrivbordsbakgrund (inget rörigt)
- [ ] **Webbläsare:** Rent Chrome-fönster, inga bokmärkesfält, inga extensions synliga
- [ ] **Zoom:** Exakt 100% i webbläsaren (Cmd/Ctrl + 0)
- [ ] **Upplösning:** 1920×1080 (Full HD)

---

## 3. Verifiera i appen

- [ ] **Inloggad:** Kontrollera att du är inloggad på bookdigest.se
- [ ] **Konto:** Reader- eller Pro-konto aktivt (alla tre stilar tillgängliga)
- [ ] **Readwise-integration:** Konfigurerad och aktiv (för export-scenen)
- [ ] **Test-upload:** Gör en snabb test-upload av Atomic Habits för att verifiera att hela flödet fungerar, ta sedan bort den innan riktiga inspelningen

---

## 4. Inspelningsinställningar

- [ ] **Skärminspelningsprogram:** QuickTime (Mac) / OBS (Windows/Mac) / Loom
- [ ] **Inställningar:** 1920×1080, 60fps
- [ ] **Mikrofon:** Klar och testad (spela in voiceover separat eller live)
- [ ] **Inspelningsyta:** Hela skärmen (inte bara webbläsarfönstret)

> **Tips:** Spela in voiceover separat i en lugn miljö — det är lättare att synka i redigering än att få rätt timing live.

---

## 5. Ordning på scenerna (lång version)

Följ scriptet i `docs/demo-video-script.md`. Grov tidplan:

```
0–12s    Scen 1: Hook (b-roll/kamera)
12–25s   Scen 2: Problem (talking head)
25–40s   Scen 3: Navigera till bookdigest.se
40–55s   Scen 4: Dra & släpp PDF
55–75s   Scen 5: Questionnaire (3 frågor)
75–90s   Scen 6: Välj stil (Executive + Action Plan)
90–110s  Scen 7: Laddningsskärm (time-lapse)
110–145s Scen 8: Scrolla through summary + sidcitationer
145–160s Scen 9: Export → Readwise
160–175s Scen 10: CTA-skärm
```

---

## 6. Efter inspelning

- [ ] Råmaterial sparat säkert (extern disk eller molntjänst)
- [ ] Lång version redigerad och exporterad (MP4, H.264, 1080p)
- [ ] Kort version klippt från lång version (60–90s)
- [ ] Thumbnail skapad (se scriptets redigeringsguide)
- [ ] Ladda upp: YouTube (lång) + Instagram/TikTok/LinkedIn (kort)
